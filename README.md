#### Functions for OpenStack Horizon
> A plugin for [Oh My Fish][omf-link].

<br/>

## Install

```fish
$ omf install horizon
```


## Usage

```fish
$ horizon_clean_workspace
$ horizon_git_review
$ horizon_reset_env
$ horizon_run_python3_tests
$ horizon_test_and_continue_js
$ horizon_test_and_continue
```
